CLEAN FLASH/First Time Installation :

  1. Download the latest build (Only vanilla build available)
  2. First Make sure recovery is installed.
  3. Reboot to recovery
  4. Make nandroid backup (Not Necessary, but just in case if you lost somewhere)
  5. Perform Factory Reset/Format
  6. Wipe system and vendor
  7. Flash latest build
  8. Wipe para ((o avoid recovery loop)
  9. Reboot to System

DIRTY FLASH :

  1. Download the latest build
  2. Reboot to recovery
  3. Flash the latest build
  4. Wipe Cache
  5. Reboot

*Note: GApps only has to be flashed once whereas Magisk/Su is to be flashed every time you Dirty Flash the OTA Update.
